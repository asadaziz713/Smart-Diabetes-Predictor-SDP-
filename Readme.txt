# Hackstorm - Smart Diabetes Predictor

### 🏥 Problem Statement
Predict diabetes using simple patient inputs with an interactive GUI.

### 💡 Our Solution
An AI-inspired scoring system that predicts diabetes risk based on user answers such as age, weight, and lifestyle.  
Built with Python + Pygame for a user-friendly interface.
